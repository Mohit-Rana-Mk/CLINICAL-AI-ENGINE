# Knowledge Base Documentation

## 1. Knowledge Source Inventory

The knowledge base contains trusted clinical guidelines and medical references collected from internationally recognized organizations.

### Sources

- World Health Organization (WHO)
- National Institute for Health and Care Excellence (NICE)
- Centers for Disease Control and Prevention (CDC)
- American Diabetes Association (ADA)
- GOLD COPD Guidelines

### Medical Specialties

- Cardiology
- Neurology
- Pediatrics
- Respiratory Medicine
- Dermatology
- Emergency Medicine
- Endocrinology
- Infectious Diseases

### File Types

- PDF Clinical Guidelines
- Medical Protocols
- Clinical Reference Documents

---

## 2. Version Control Policy

- Every document must be added through Git.
- Existing medical documents should never be overwritten without verification.
- New documents must follow the existing folder structure.
- File names should remain descriptive and consistent.
- All changes should be committed with meaningful commit messages.

---

## 3. Knowledge Base Update Workflow

1. Collect trusted medical documents.
2. Validate document authenticity.
3. Organize documents into specialty folders.
4. Update the document catalog.
5. Validate file names and metadata.
6. Run the indexing pipeline.
7. Commit and push changes to the repository.

---

## 4. Validation Checklist

Before adding a document, verify:

- Source is trusted.
- PDF opens correctly.
- File name is meaningful.
- Correct specialty folder is used.
- No duplicate documents exist.
- Document catalog is updated.
- Indexing pipeline completes successfully.

---

## 5. QA Methodology

Quality assurance follows these principles:

- Only trusted medical organizations are used.
- Questions are based on official clinical guidelines.
- Every QA entry contains:
  - Question
  - Expected Source
  - Disease
  - Specialty
  - Difficulty
- Duplicate questions are avoided.
- Dataset quality is reviewed before indexing.

---

## Summary

The medical knowledge base is designed to maintain a reliable, organized, and validated collection of clinical references that support the AI Clinical Decision Support Engine.
